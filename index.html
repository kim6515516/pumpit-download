🎮 사용 예시
Steam PUMP IT UP RISE는 키보드 입력만 지원해서 가정용 발판을 그대로 연결할 수 없습니다. 이 프로그램을 켜두면 발판 입력이 키보드 입력으로 바뀌어 PUMP IT UP RISE에서 그대로 인식됩니다.
사용법
1. 발판 USB 연결 → 2. PIU_Mapper.exe 실행 → 3. '▶ 시작' 클릭
패널 버튼을 클릭하면 원하는 키로 매핑을 변경할 수 있습니다.

VID 0x0D2F / PID 0x0001 · 국내 정발 가정용 발판
